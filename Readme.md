# Fraud Detection on Bank Payments

## Overview
Machine learning-based fraud detection system built using Random Forest and XGBoost.

## Dataset
Banksim dataset (synthetic bank payment transactions)

## Techniques Used
- Data preprocessing and encoding
- SMOTE for class imbalance
- Random Forest and XGBoost models
- ROC-AUC evaluation

## Tools
Python, Pandas, Scikit-learn, XGBoost
